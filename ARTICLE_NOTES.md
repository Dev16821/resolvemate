# ResolveMate Article Notes

## Core Story
Most support agents respond like they have amnesia. ResolveMate uses memory to remember previous customer complaints, rejected solutions, and preferred tone.

## Main Technical Idea
The agent changes its response based on recalled customer memory.

## Before Memory
The response is generic because no previous customer context exists.

## After Memory
The response becomes more context-aware because the agent remembers:
- previous issue
- rejected solution
- preferred resolution
- response tone

## Demo Example
Customer: Ravi Kumar

Previous issue:
Wrong food order delivered

Rejected solution:
Coupon

Preferred resolution:
Refund-focused support

New issue:
Late delivery

Behavior:
ResolveMate avoids suggesting coupons and drafts a more direct response.
