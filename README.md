# resolvemate


## Problem
Customer support teams often forget previous customer complaints, rejected solutions, and customer preferences. This leads to generic responses and repeated customer frustration.

## Solution
ResolveMate is an AI customer support agent that uses Hindsight to remember customers’ past interactions and generate more context-aware responses.

## Hindsight Usage
The agent stores customer complaints, resolutions, rejected solutions, preferences, and response tone requirements.

## Demo Behavior
### First Interaction
The agent gives a generic support response.

### Later Interaction
The agent recalls previous customer history and generates a personalized response based on memory.
